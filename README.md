# WishLinker

`Análise e Desenvolvimento de Sistemas - PUC Minas`

`Desenvolvimento de aplicação web front-end`

`2º semestre/2023`

O objetivo geral deste projeto é oferecer aos consumidores uma plataforma de centralização de produtos de interesse. Através dessa plataforma, os consumidores poderão tomar melhores decisões de compra.
